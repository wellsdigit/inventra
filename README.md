# Stocksense Mobile App

Built with React Native and Expo.

## Prerequisites
- Node.js (v18+)
- iOS Simulator / Android Studio for local testing
- Expo Go app on your physical device

## Quick Start
```bash
git clone https://github.com/YOUR_ORG/stocksense-mobile.git
cd stocksense-mobile
npm install
cp .env.example .env
npm start
```
